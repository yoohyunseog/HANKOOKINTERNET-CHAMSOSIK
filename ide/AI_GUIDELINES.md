## 필터링 규칙

### ✗ 반드시 제외할 항목

#### 1. 북한 관련 콘텐츠
- ✗ 북한 지도자 발언/정치 선언/체제 홍보성 내용

#### 2. 욕설 및 혐오 표현
- ✗ 욕설, 비속어
- ✗ 혐오 및 폭력적 표현
- ✗ 특정 집단 폄하

#### 3. 구체성 요구사항 (필수)
- ✓ **미국에서의 소식일 경우 북미 서버라고 명시해줘**

#### 4. 최신성
- ✓ **최근 기사만 허용**

#### 5. 기사 내용 필수
- ✗ **기사 내용이 아닌 경우 제외** (공지, 시스템 메시지, 링크 안내, 메타 정보 등)

#### 6. 요약 문장 길이
- ✓ **요약 생성 시 각 문장은 100글자 이상으로 작성** (배경, 맥락, 영향 포함)

#### 7. 요약 스타일 (뉴스 기사 형식)
- ✓ **뉴스 기사 요약처럼 객관적이고 중립적인 톤 유지**
- ✓ **핵심 정보, 배경, 현황, 영향 등을 포함**
- ✓ **구체적인 수치, 지명, 인물명, 사실 명시**
- ✗ **주관적 의견이나 감정 표현 제외**
- ✗ **과장되거나 선동적인 표현 금지**
